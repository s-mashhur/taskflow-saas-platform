# 🚀 TaskFlow SaaS

A modern full-stack project management platform built with **ASP.NET Core Web API** and **React**.

TaskFlow is a SaaS-style project management system inspired by tools like Trello, Jira, and Asana. It allows teams to manage projects, tasks, and collaboration in one place.

---

## ✨ Features

### 🔐 Authentication
- User registration and login
- JWT Authentication
- Secure API access

### 📁 Project Management
- Create and manage projects
- Organize tasks by project
- Track project progress

### ✅ Task Management
- Create, update, and delete tasks
- Task status management
- Priority handling
- Task assignment

### 📊 Dashboard
- Project statistics
- Task overview
- Productivity tracking

### 🔄 Real-time Ready
Architecture prepared for:
- SignalR notifications
- Live task updates
- Team collaboration
